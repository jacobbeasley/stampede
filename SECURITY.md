# Security Policy

## Supported Versions

We actively support and patch the latest release of this project.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Security Updates and Vulnerabilities

We take security seriously. To maintain the security of this project:
- We will patch known vulnerabilities as soon as they are identified and verified.
- We perform minor and patch version updates of our dependencies as needed to address bugs, performance issues, and security vulnerabilities. Dependabot is running and patching regularly (and running automated functional and unit test after each patching cycle to ensure there are no regressions).

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, report it using one of the following methods:
- Open a draft Security Advisory directly in the GitHub repository.
- Alternatively, contact the maintainers.

We will investigate all reports and address them promptly.
