# DEV-2: Automated Testing

## Description
Write Playwright automated tests that set up a test admin user, open up a web browser, login, and logout. These tests could execute on every future jules session.

## Acceptance Criteria
- Playwright is installed and configured.
- A global setup script is created to set up a test admin user in the database (or through the app).
- A test is written to open the browser, navigate to the app, log in as the admin user, and log out.
- The tests run successfully against the local development server.
