# DEV-1: Scaffold Repository — Implementation Plan

## Goal

Initialize a working Buffalo v0.18 application in the repo root, replace the default Webpack frontend pipeline with Vite + Svelte + Tailwind CSS + DaisyUI, and wire everything together so `buffalo dev` + `npm run dev` gives a hot-reloading full-stack development environment.

## Decisions Made

| Decision | Choice |
|---|---|
| Database | PostgreSQL |
| Go module | `buffalo-app` |
| Frontend build | Vite (skip-webpack) |
| Tailwind version | v4 (via `@tailwindcss/vite`) |
| DaisyUI version | v5 |
| Svelte version | v5 (Runes syntax) |
